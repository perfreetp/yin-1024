## 1. 产品概述

「同居记」是一款面向异地合租年轻人的移动端协作应用，专注解决公共用品补货、家务协商和共同开销透明化三大痛点。通过将琐碎协作融入日常操作，让室友之间少扯皮、多留痕，营造和谐合租生活。

- 目标用户：18-35岁合租年轻人，特别是异地工作、初入社会的都市青年
- 核心价值：消除合租中的沟通摩擦和信息不对称，建立可追溯的协作机制

## 2. 核心功能

### 2.1 用户角色

| 角色 | 加入方式 | 核心权限 |
|------|----------|----------|
| 屋主 | 创建合租屋 | 管理房规、邀请/移除室友、设置轮值规则 |
| 室友 | 邀请码加入 | 参与家务、购物、记账、投票等全部协作功能 |

### 2.2 功能模块

1. **首页**：今日概览、待办提醒、快捷入口、月末总结卡片
2. **家务板**：轮值排班、请假换班、清洁打卡、房规置顶
3. **购物清单**：共编清单、缺货拍照补录、购买状态追踪
4. **账本**：价格分摊、垫付认领、账单提醒、公共基金余额
5. **冰箱页**：食材标记、临期提醒、共享物品借用
6. **群聊**：群内公告、日常沟通
7. **投票页**：安静时段投票、公共事务决策
8. **访客管理**：访客报备、来访记录
9. **个人中心**：合租屋管理、室友邀请、个人设置

## 3. 核心流程

### 3.1 合租屋创建与加入流程
用户打开应用 → 创建合租屋（输入屋名、地址） → 生成邀请码 → 分享给室友 → 室友输入邀请码加入 → 开始协作

### 3.2 家务轮值与打卡流程
屋主设置轮值规则 → 系统自动生成排班表 → 值日生收到提醒 → 完成家务后打卡 → 室友可见打卡记录 → 需要请假时发起换班申请

### 3.3 购物与记账流程
发现缺货 → 拍照补录到清单 → 室友补充需求 → 某人购买 → 标记已购并录入金额 → 系统自动分摊 → 垫付人生成收款提醒 → 室友确认并结算

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖珊瑚色 (#FF6B6B) + 奶油白 (#FFF8F0)，传递温暖居家感
- **辅色调**：薄荷绿 (#4ECDC4) 用于成功状态，柔和紫 (#A78BFA) 用于投票/决策
- **按钮风格**：圆角胶囊按钮，微阴影，点击时缩放反馈
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- **布局风格**：移动端底部 Tab 导航，卡片式内容布局，圆角卡片带柔和阴影
- **图标风格**：线条图标 (Lucide)，2px 描边，与整体温暖风格搭配
- **动效**：页面切换滑动过渡，列表项入场错位动画，按钮弹性反馈

### 4.3 响应式设计

- **移动优先设计**：所有页面以 375px 宽度为基准设计，最大宽度 430px 居中展示
- **触摸优化**：按钮最小点击区域 44px，列表项左滑/右滑手势操作
- **底部导航**：5个主要入口（首页/家务/购物/账本/我的），中间突出打卡按钮
- **二级页面**：从右侧滑入，左上角返回按钮
